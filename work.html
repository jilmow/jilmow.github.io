MicroJobs.sr

MicroJobs.sr was a group project that I worked on with other students at UNASAT.

The website was created for two types of users: people searching for jobs and people or businesses offering jobs.

My contribution: Front-end development

Technologies used: HTML, CSS and JavaScript

GRASI

GRASI was also a group project that I worked on with other students at UNASAT.

The goal was to make a self-working smart grass cutter. With sensors, cameras and a navigation system etc. to cut the grass automatically.

My contribution: Programming the sensors and camera to detect obstacles and cut the grass

Technologies used: Arduino, C++ and Python

Delivery Bot (Hackomation)

This was a group project that I worked on while studying at Anton de Kom University of Suriname.

Our goal was to create a robot that could deliver packages from one location to another. My responsibility was creating the code for the navigation system.

My contribution: Navigation and live location programming

Project type: Group project
